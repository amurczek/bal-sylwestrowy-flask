# Projekt zaliczeniowy – Flask

Aplikacja webowa do zapisów na Bal Sylwestrowy (DoubleTree by Hilton Warszawa).

Użytkownik może:
- zobaczyć informacje o wydarzeniu
- wypełnić formularz zgłoszeniowy

Administrator może:
- zobaczyć listę zapisanych osób
- usuwać zgłoszenia
- potwierdzać rezerwację

Technologie:
- Python
- Flask
- SQLite
- HTML (Jinja2)
- CSS

Uruchomienie:
python app.py
